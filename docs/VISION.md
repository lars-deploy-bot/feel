# Vision

## The Game

You're running a startup. But your employees are AI agents.

Hire them. Assign roles. Watch your company run. Tweak. Optimize. Scale.

Rollercoaster Tycoon, but the rides are workflows and the guests are customers.

## The Fun

Building the machine:
- Hire a "frontend dev" agent
- Hire a "code reviewer" agent
- Hire a "customer support" agent
- Wire them together into workflows

Watching it run:
- Customer submits bug report
- Support agent triages
- Dev agent fixes
- Reviewer agent approves
- Customer gets update

You didn't do any of that. You built the system that did.

## Why It Works

**Git worktrees** - Every agent works in isolation. Nothing breaks prod. Merge when ready.

**Pre-configured roles** - Agents aren't figuring it out. They have job descriptions. They do their job.

**Predictable execution** - Same input, same output. Every time. That's what makes it a game, not a gamble.

## The Feeling

You wake up. Check your startup dashboard.

3 bugs fixed overnight. 2 features shipped. 14 customer tickets resolved.

You didn't write any code. You built the team that did.

That's the feeling.

## The Constraint

**Your startup has to actually work.**

Not "impressive demo." Not "watch the AI try."

Real customers. Real code. Real money.

That means:
- Agents do exactly what their role says
- Agents know when to escalate, not guess
- Small scope per agent = predictable outcomes
- The system works because each part works

## What We're Building

A game where you manage a startup powered by AI agents.

The game is real. The startup is real. The agents ship real code to real customers.

And it just works.

## Why This Doesn't Exist Yet

**"AI Employee" platforms exist:**
Lindy, Relevance AI, Ema, Sintra - all pitch "hire AI employees." But they're enterprise dashboards. Boring. Complex. Not fun.

**Multi-agent orchestration exists:**
CrewAI, Microsoft Agent Framework, OpenAI AgentKit - all coordinate multiple agents. But they're developer tools. Infrastructure. Not a product.

**AI tycoon games exist:**
"My AI Company: Startup Tycoon", "AI Tycoon" - casual mobile games about building an AI company. Simulated. Fake. Not real agents doing real work.
